Workspace ONE UEM QR Code Helper - User Guide

Step-by-Step Guide

1. Enable WiFi Configuration (Optional)

If you want to include WiFi details in the QR code, check the "Enable WiFi Configuration" checkbox. Then, provide the following details:

  • WiFi SSID: Enter the WiFi network name (SSID).
  • WiFi Password: Enter the WiFi password.

2. Fill in Device Information

Fill in the following fields to provide the necessary information for device provisioning:

  • Device Server: Enter the Workspace ONE UEM device server URL.
  • Group ID: Enter your organization's Group ID.
  • Username: Enter the username for device enrollment.
  • Password: Enter the password for the username.

3. Configure Additional Settings

Configure extra settings as needed:

  • useUEMAuthentication: Check this box to enable UEM authentication in the QR code. The value will be added as "true" or "false" in the JSON.
  • Enable System Apps: Keep this box checked to leave all system apps enabled during provisioning.

4. Generate QR Code

Click the "Generate QR Code" button to generate a QR code based on the details you've entered. The QR code will appear on the screen, along with an editable JSON representation of the data.

5. View and Edit JSON

The tool displays the JSON used to generate the QR code. If needed, edit the JSON directly in the text box to customize the QR code.

6. Save JSON

After editing the JSON, click the "Save JSON" button to update the JSON. This will not regenerate the QR code until you click "Regenerate QR Code".

7. Regenerate QR Code

If you’ve modified and saved the JSON, click the "Regenerate QR Code" button to generate a new QR code based on the updated JSON data.

Is Text/QR code processed & stored on remote server?
No, all the processing is done on your browser engine, and is limited to this current session. No information is retained or stored.
